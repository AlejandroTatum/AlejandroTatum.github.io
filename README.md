# Alejandro Padilla Portfolio

Personal portfolio built with Next.js, TypeScript and Tailwind CSS.

## Features

- Terminal-inspired dark UI
- Responsive layout
- Featured academic projects
- Remote opportunity positioning
- GitHub Pages deployment

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- GitHub Actions
- GitHub Pages

## Local development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```
